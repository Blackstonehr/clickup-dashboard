STEP-BY-STEP INSTRUCTIONS:
================================================================================

STEP 1: Download
───────────────
☑ Click the BIG GREEN "LTS" button
☑ Wait for download to complete (takes about 30 seconds)
☑ You'll see a file named: node-v20.x.x-x64.msi

STEP 2: Install
───────────────
☑ Open your Downloads folder
☑ Double-click the .msi file you just downloaded
☑ Click "Next" on the first screen
☑ Click "I accept the agreement" then "Next"
☑ Click "Next" (keep default location)
☑ Click "Next" (keep default components)
☑ ✅ CHECK "Automatically install the necessary tools"
☑ Click "Install"
☑ Wait 1-2 minutes for installation
☑ Click "Finish" when done

STEP 3: Restart
───────────────
⚠️  THIS IS CRITICAL: RESTART YOUR COMPUTER NOW!

☑ Click Start button
☑ Click Power button
☑ Click Restart
☑ Wait for computer to restart

STEP 4: Verify
───────────────
☑ Open PowerShell (search for "PowerShell" in Start menu)
☑ Type: node --version
☑ Press Enter
☑ You should see: v20.x.x
☑ Type: npm --version
☑ Press Enter
☑ You should see: 10.x.x

✅ If you see both version numbers, SUCCESS!

================================================================================

Come back to this folder and run:

   powershell -ExecutionPolicy Bypass -File setup-helper.ps1

2. Install project dependencies
3. Help you configure ClickUp credentials
4. Start the dashboard!

================================================================================
TROUBLESHOOTING:
================================================================================

Problem: "Can't find the download button"
Solution: Look for big green button that says "LTS"

Problem: "Download won't start"
Solution: Try this direct link: https://nodejs.org/dist/v20.11.0/node-v20.11.0-x64.msi

Problem: "Installer won't run"
Solution: Right-click the file → Run as administrator

Problem: "Still says node not recognized"
Solution: Did you RESTART? Try closing and reopening PowerShell

Problem: "Website is in a different language"
Solution: Scroll to the bottom, look for English (EN) link
